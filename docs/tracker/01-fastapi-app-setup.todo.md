# 01: FastAPI App Setup

## Description
Set up the FastAPI application skeleton with a health check endpoint and CLI command to start the server.

## Scope
- `backend/codehive/api/app.py` — FastAPI app factory
- `backend/codehive/config.py` — Settings via environment variables (host, port, debug)
- `backend/codehive/cli.py` — Replace argparse stub with `codehive serve` command
- `backend/tests/test_api.py` — Health check tests

## Dependencies
- None (first issue)
